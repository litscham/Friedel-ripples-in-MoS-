# Computational: Pristine MoS₂ Reference System

## Objective
Calculate and validate defect-free monolayer MoS₂

## Tasks
- Construct 2D supercell for monolayer MoS₂
- Optimize atomic geometry
- Calculate band structure and compare with literature
- Compute density of states (DOS)
- Generate LDOS maps at different energies

## Deliverable
Reference system data: band structure plots, DOS figures, validated geometry
