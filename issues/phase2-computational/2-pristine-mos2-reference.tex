\section{Objective}
Calculate and validate defect-free monolayer MoS\textsubscript{2} electronic structure as baseline for STEM-EELS comparison.

\section{Tasks}
\begin{itemize}
    \item Construct 2D supercell for monolayer MoS\textsubscript{2}
    \item Optimize atomic geometry
    \item Calculate band structure and compare with literature
    \item Compute density of states (DOS)
    \item Generate LDOS maps at different energies
\end{itemize}

\section{Deliverable}
Reference system data: band structure plots, DOS figures, validated geometry, and baseline LDOS maps.
