\section{Objective}
Study Mo-S and S-Mo antisite defects with detailed LDOS characterization for STEM-EELS signatures.

\section{Tasks}
\begin{itemize}
    \item Create supercells with antisite substitutions
    \item Optimize geometries with symmetry variations
    \item Calculate band structures and DOS
    \item Map LDOS for both defect types
    \item Analyze hybridization effects
    \item Compare with vacancy-induced changes
\end{itemize}

\section{Deliverable}
Antisite defect results: structures, electronic properties, LDOS visualizations, and hybridization analysis.
